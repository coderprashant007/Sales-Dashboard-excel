# Sales Dashboard (Excel)

## Project Overview
This project is an interactive Sales Dashboard created in Excel to analyze sales performance and generate business insights.

## Key Features
- Month-wise sales analysis
- Gender-wise and age-wise sales insights
- Channel-wise performance (Amazon, Myntra, Flipkart)
- State-wise sales analysis
- Top 5 customers identification
- Order status tracking (Delivered, Cancelled, Returned)

## Tools Used
- Microsoft Excel
- Pivot Tables
- Power Query
- Charts & Slicers

## Key Insights
- Identified top-performing sales channels and states
- Analyzed customer buying patterns
- Helped in decision-making and performance tracking
